How to build your own Spotify Track Saver ?

Build your webhook Save your tracks
  1. Go on the IFTTT website and create an account.
  2. Create a new Applet.
  3. This
    • Select Webhooks.
    • Select Receive a web request.
  4. That
    • Select Spotify.
    • Log in to your Spotify account.
    • Select Save a track.
  5. Then
    • Add Value1 to Search query and Value2 to Artist name.
    • Enter the Event Name track_saver to create the trigger.
    • Click on Finish.
  1. Now that your webhook is ready, you can either:
    • Use the Track Saver provided on this website.
    • Create your own Track Saver.
    In both cases, you will need your IFTTT Key: Get it here (in the Documentation tab).
  2. To create your own Track Saver:
    • Remix this project to Glitch.
    • Add your IFTTT Key in the javascript file.
    • Customize it as you want.
